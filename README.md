
# AI Hacker Assistant

ИИ-агент для обучения и автоматизации этичного пентеста.

## Требования
- Python 3.8+
- Ollama (или другой OpenAI-совместимый сервер)
- (Опционально) Kali Linux с установленными инструментами

## Установка
```bash
pip install -r requirements.txt
```

### Структура проекта:
```
ai-hacker-assistant/
├── .venv/..
├── core/                     ← Ядро (платформо- и UI-независимое)
│   ├── ai_client.py          ← Работа с ИИ (API, форматы, история чата)
│   ├── command_executor.py   ← Безопасное выполнение команд
│   ├── tools/                ← Модули для nmap, sqlmap, metasploit и т.д.
│   └── config.py             ← Загрузка настроек (API ключ, модель, ОС)
│
├── interfaces/               ← Интерфейсы (можно добавлять новые)
│   ├── cli.py                ← CLI-интерфейс (запуск через терминал)
│   └── gui/                  ← Папка для будущего GUI (Tkinter/PyQt)
│       └── main_window.py
│
├── platforms/                ← Платформо-специфичная логика
│   ├── windows.py            ← Упаковка KLT? Позже. Сейчас — предупреждение.
│   └── kali.py               ← Проверка наличия инструментов (nmap и др.)
│
├── main.py                   ← Точка входа: определяет ОС → запускает нужный интерфейс
├── requirements.txt
└── README.md
```
